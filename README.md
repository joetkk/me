# me
My Portfolio

This is my personal portfolio website showcasing my projects and skills.

## Viewing the Portfolio

To view the portfolio locally:
1. Open `index.html` in your web browser.

## Hosting on GitHub Pages

This repository is set up to be hosted on GitHub Pages. To enable:
1. Go to your repository settings
2. Scroll to "Pages" section
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Save

Your portfolio will be available at `https://joetkk.github.io/me/`

## Customization

- Edit `index.html` to update content (CSS is included inline)
- Add images or other assets as needed
